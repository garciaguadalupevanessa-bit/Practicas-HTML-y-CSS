# Practicas-HTML-y-CSS
Ejercicios prácticos en HTML y CSS

Para crear un CV HTML y usar CSS para el estilo del mismo.

Muy importante el nombre:  "nombre".html debe estar en minúsculas, para evitar que de error al sincronizar la parte de VSC de código y la visualización en el archivo.

# 📄 Proyecto: Mi CV en HTML y CSS

Este repositorio contiene el código fuente para mi currículum profesional en formato web. El objetivo es crear una estructura limpia, moderna y fácil de leer tanto para humanos como para algoritmos.

---

## 🛠️ Cómo crear el archivo (Paso a Paso)

### 1. Estructura de Archivos
En Visual Studio Code, he creado la siguiente estructura:
* `index.html`: El esqueleto (títulos, secciones, textos).
* `style.css`: El diseño (colores, fuentes, márgenes).

### 2. El código HTML (index.html)
He utilizado etiquetas semánticas para que los buscadores entiendan mi perfil:
- `<header>`: Para mi nombre y contacto.
- `<section>`: Para experiencia y educación.
- `<ul>`: Para listar mis habilidades en **Python** e **IA**.

### 3. El diseño CSS (style.css)
Para que el CV se vea profesional, he configurado:
* **Fuentes:** Una letra tipo *Sans-serif* para mayor claridad.
* **Colores:** Un tono azul oscuro (`#2c3e50`) para los títulos, que transmite confianza.
* **Responsive:** El diseño se adapta si se ve desde un móvil.

---

## 🚀 Instrucciones para subir el CV

Para publicar este CV y que sea accesible, sigue estas indicaciones:

1. **Previsualización:** Abre el archivo `index.html` en tu navegador para revisar que todo esté en su sitio.
2. **Exportar a PDF:** - Abre el HTML en Chrome.
   - Presiona `Ctrl + P` (Imprimir).
   - Selecciona **"Guardar como PDF"**.
3. **Subida a GitHub:**
   - Sube tanto el código (`.html` y `.css`) como el `.pdf` generado a la carpeta `/docs`.

---

## 💡 Fundamentos aprendidos
* **Estructura lógica:** Organizar la información de lo más importante a lo menos importante.
* **Estética funcional:** El diseño no es solo "que sea bonito", es que sea fácil de escanear visualmente.
